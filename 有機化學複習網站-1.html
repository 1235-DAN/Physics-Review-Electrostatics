01

核心概念概述

💡
本章主軸：烴類（Hydrocarbons）
本章圍繞只含碳（C）與氫（H）的有機化合物展開。根據碳碳鍵的種類，烴類分為飽和烴（烷、環烷）與不飽和烴（烯、炔、芳香烴）。飽和烴碳碳間只有單鍵，化性穩定；不飽和烴含 π 鍵，化性活潑，容易發生加成反應。芳香烴（苯系）雖屬不飽和烴，但因 π 鍵形成共振（離域），結構特別穩定，主要發生取代而非加成反應。
🔑
結構決定性質
有機化合物的「官能基」決定化學活性。碳碳雙鍵（烯基）讓烯類容易被溴水、KMnO₄ 氧化；碳碳參鍵讓炔類可連續加成。物理性質上，碳數越多分子量越大，沸點越高；所有烴類不溶於水，可溶於非極性有機溶劑（苯、CCl₄ 等）。
⛓
飽和烴
CₙH₂ₙ₊₂ / CₙH₂ₙ
碳碳間全為單鍵（σ鍵），包含鏈烷與環烷。化性穩定，常溫下幾乎不反應，可燃燒。
烷 環烷 σ鍵
⚡
不飽和烴（脂肪烴）
CₙH₂ₙ / CₙH₂ₙ₋₂
含 C=C 或 C≡C，π 鍵弱，化性活潑。可使溴水及 KMnO₄ 褪色，進行加成與聚合反應。
烯 炔 π鍵
🌸
芳香烴（苯系）
C₆H₅— / CₙH₂ₙ₋₆
含苯環，π 電子共振離域，結構穩定，鍵長均等（1.39 Å）。主要進行取代，不易加成。
苯 共振 取代反應
02

官能基分類

🔵
烷（Alkane）
C–C 單鍵，無官能基
結構：全為 σ 鍵，非極性，最穩定的烴類。
物性：不溶於水，比重 < 1，C₁~C₄ 為氣體。
反應：可燃燒；鹵化需光照或高溫（取代）。
例：甲烷 CH₄、乙烷 C₂H₆
燃燒 取代 飽和
🟢
烯（Alkene）
C=C 雙鍵（烯基）
結構：含 1 個 π 鍵 + 1 個 σ 鍵，平面形（120°）。
物性：不溶於水，化性活潑。
反應：使溴水褪色（加成）；使 KMnO₄ 褪色（氧化）；可聚合。
例：乙烯 CH₂=CH₂、丙烯 C₃H₆
加成 聚合 氧化 褪色溴水✓
🟡
炔（Alkyne）
C≡C 參鍵（炔基）
結構：含 2 個 π 鍵 + 1 個 σ 鍵，線形（180°）。
物性：不溶於水，反應性比烯更強。
反應：連續加成；末端炔（-C≡C-H）可與銀氨液生成白色沉澱。
例：乙炔 C₂H₂、丙炔 C₃H₄
加成 末端取代 銀氨液→白↓
🔮
芳香烴（Arene）
苯環（共振結構）
結構：6 個碳平面六角形，鍵長均等 1.39 Å，鍵級 1.5。
物性：不溶於水，有特殊氣味，具共振穩定性。
反應：主要為取代（鹵化、硝化、磺化）；不使溴水褪色。
例：苯 C₆H₆、甲苯 C₇H₈、萘 C₁₀H₈
取代 共振穩定 溴水不褪色
03

有機反應總覽

① 烷類反應

類型方程式條件⭐ 考試重點
燃燒 CH₄ + 2O₂ → CO₂ + 2H₂O 充足 O₂，點火 最重要能源用途；ΔH = -890 kJ/mol
鹵化取代 CH₄ + Cl₂ →(hν) CH₃Cl + HCl 光照或高溫（非催化劑！） ⭐ 自由基機制；可連續取代到 CCl₄
硝化取代 CH₄ + HNO₃ →(>475°C) CH₃NO₂ + H₂O 高溫（>475°C），濃硝酸 ⭐ H 被 -NO₂（硝基）取代，生成硝基甲烷；與鹵化同為取代反應
裂解 C₄H₁₀ → C₃H₆ + CH₄ 高溫、隔絕空氣、觸媒 ⭐ 工業製烯烴的方法
脫氫 C₄H₁₀ → C₄H₈ + H₂ 高溫、觸媒 脫氫後形成烯類；鏈不斷裂

② 烯類反應

類型方程式條件⭐ 考試重點
氫化加成 CH₂=CH₂ + H₂ →(Pt) CH₃CH₃ Pt 或 Ni 催化 ⭐ 食用油氫化固化；反式脂肪來源
鹵化加成 CH₂=CH₂ + Br₂ → CH₂BrCH₂Br 直接混合，無需催化 ⭐ 溴的紅棕色褪色 → 鑑別烯類！
HX 加成 CH₂=CH₂ + HBr → CH₃CH₂Br 馬可尼可夫法則 ⭐ H 加氫多碳，X 加氫少碳
水合加成 CH₂=CH₂ + H₂O →(H₂SO₄) C₂H₅OH H₂SO₄ 催化，加熱 ⭐ 工業製乙醇的方法
聚合 nCH₂=CH₂ → -(CH₂CH₂)ₙ-（PE） 高溫高壓 + 催化劑 ⭐ 乙烯→PE，氯乙烯→PVC，苯乙烯→PS
KMnO₄ 氧化 烯 + KMnO₄ → 二元醇 + MnO₂↓ 中性或微鹼性，室溫 ⭐ 紫色褪色 + 黑色沉澱 → 鑑別烯!
醇脫水製烯 C₂H₅OH →(H₂SO₄,170℃) CH₂=CH₂ + H₂O 濃 H₂SO₄，170~180°C ⭐ 實驗室製乙烯的標準方法

③ 炔類反應

類型方程式條件⭐ 考試重點
氫化（1步） HC≡CH + H₂ →(Pt) CH₂=CH₂ 控制 H₂ 量 可停在烯的階段
鹵化加成 HC≡CH + Br₂ → CHBr=CHBr →(+Br₂)→ CHBr₂CHBr₂ 直接混合，可兩步 ⭐ 溴的顏色由紅棕→無色；鑑別炔
水合 HC≡CH + H₂O →(HgSO₄) [CH₂=CHOH] → CH₃CHO（乙醛）
CH₃-C≡CH + H₂O →(HgSO₄) [CH₃-C(OH)=CH₂] → CH₃COCH₃（丙酮） 		HgSO₄ / H₂SO₄ 催化，加熱 ⭐ 末端炔水合：乙炔→乙醛；丙炔→丙酮（非末端碳加 OH，中間體烯醇重排為酮）。絕非乙醇！
三聚（製苯） 3HC≡CH →(500°C) C₆H₆（苯） 約 500°C，石英管 ⭐ 乙炔 3 分子聚合成苯！
KMnO₄ 氧化 3HC≡CH + 10KMnO₄ → 6CO₂ + 10MnO₂↓ 中性或微鹼性 褪色並產生 CO₂（烯的產物是二元醇）
末端炔取代 HC≡CH + 2[Ag(NH₃)₂]⁺ → Ag-C≡C-Ag↓（白色） 銀氨溶液，室溫 ⭐ 只有末端炔才有！R-C≡C-H 的鑑別關鍵
電石製乙炔 CaC₂ + 2H₂O → HC≡CH + Ca(OH)₂ 室溫，CaC₂（電石）加水 ⭐ 實驗室製乙炔；CaC₂ = 電石

④ 芳香烴（苯）反應

類型方程式條件⭐ 考試重點
鹵化取代 苯 + Cl₂ →(FeCl₃) 氯苯 + HCl 
  ⬡  +  Cl₂  →(FeCl₃)  ⬡-Cl  +  HCl

 FeCl₃ 或 Fe 催化 ⭐ 苯主要進行取代，不是加成！需催化劑
硝化取代 苯 + HNO₃ →(H₂SO₄,△) 硝基苯（C₆H₅NO₂）+ H₂O
甲苯三硝化 → TNT（2,4,6-三硝基甲苯）：
甲苯（C₆H₄(CH₃)）+ 3HNO₃ →(H₂SO₄,△) 2,4,6-三硝基甲苯 + 3H₂O
結構：苯環上 CH₃ 在 1 位，NO₂ 分別在 2、4、6 位 
        CH₃
         |
   O₂N-⬡-NO₂    （苯環，CH₃在頂，NO₂在2,4,6位）
         |
        NO₂
俗稱「黃色炸藥（TNT）」
 濃 HNO₃ + 濃 H₂SO₄，加熱 ⭐ 苯→硝基苯（一硝化）；甲苯三次硝化→TNT。硝化常考！
磺化取代 苯 + H₂SO₄ →(△) 苯磺酸 + H₂O 濃 H₂SO₄，高溫 了解磺化類型即可
烷基化 苯 + CH₃Cl →(AlCl₃) 甲苯 + HCl AlCl₃ 催化劑 ⭐ Friedel-Crafts；製備脂芳烴
側鏈氧化 甲苯 + KMnO₄/OH⁻/△ → 苯甲酸（C₆H₅COOH） 
  ⬡-CH₃  →(KMnO₄/OH⁻/△)  ⬡-COOH
  甲苯                      苯甲酸

  ⬡-CH₂CH₃  →(KMnO₄)  ⬡-COOH + CO₂
  乙苯                   苯甲酸（側鏈不管幾個碳都氧化成 -COOH）
 KMnO₄ 鹼性，加熱 ⭐ 苯環不被氧化，側鏈氧化成 -COOH；甲苯、乙苯→苯甲酸

命名規則

IUPAC 命名五步驟

1
找最長碳鏈（含官能基）
選含官能基（如 C=C、C≡C）的最長碳鏈為主鏈；若有同等長度，選含最多取代基者。
2
從接近取代基端開始編號
讓取代基（或官能基）的位置號碼盡可能最小。
3
標示取代基位置與名稱
用數字標位置，數字與名稱間用「-」連接，例：2-甲基。
4
多個相同取代基加「二、三……」
例：2,2-二甲基；位置號之間以「,」隔開。
5
寫出完整名稱
格式：[位號-個數-取代基名] + [主鏈名]，例：2,2-二甲基丁烷。

📐 異構物（Isomers）觀念整理

🔑
定義
異構物（同分異構物）：分子式相同，但結構式不同的化合物。分子式相同≠化性相同，需看結構。
① 碳鏈異構物
碳鏈的連接方式不同。
例：C₅H₁₂
正戊烷：C-C-C-C-C
異戊烷：C-C(C)-C-C
新戊烷：C-C(C)(C)-C
② 位置異構物
官能基相同，但接在不同位置。
例：C₃H₈O（醇）
1-丙醇：CH₃CH₂CH₂OH
2-丙醇：CH₃CH(OH)CH₃
③ 官能基異構物
分子式相同但官能基種類不同。
例：C₂H₆O
乙醇：CH₃CH₂–OH（醇）
甲醚：CH₃–O–CH₃（醚）
④ 幾何異構物（順反）
雙鍵無法旋轉，兩端碳各接不同基才存在。
例：2-丁烯
順-2-丁烯：兩 CH₃ 在同側
反-2-丁烯：兩 CH₃ 在異側
條件：C=C 兩端各接不同原子團
⚠️
同系物 vs 異構物 易混淆！
同系物：官能基相同，且分子式相差 CH₂ 整數倍，化性相似。（例：甲烷、乙烷、丙烷）
異構物：分子式完全相同，但結構不同，化性可能截然不同。（例：乙醇 vs 甲醚，同為 C₂H₆O）
⚠️ 乙烯（C₂H₄）與環丙烷（C₃H₆）分子式不同 → 不是異構物，也不是同系物！
05

學測／指考練習題

👆 點擊題目展開解答

Q1
【官能基判斷】下列化合物中，哪些具有「順反異構物（幾何異構物）」？ (A) CH₂=CH₂ (B) CH₃CH=CHCH₃ (C) CH₂=C(CH₃)₂ (D) ClCH=CHCl (E) CH₂=CHCl
▼
ANSWER 正確答案：(B)(D) 順反異構物存在的條件：雙鍵兩端的碳各接有「不同的兩個原子（團）」。 (A) 乙烯：每個碳都接兩個 H → 相同 → ✗ (B) 2-丁烯：每碳分別接 H 和 CH₃，兩端都不同 → ✓ (C) 2-甲基丙烯：其中一碳接兩個 CH₃ → 相同 → ✗ (D) 1,2-二氯乙烯：每碳接 H 和 Cl，不同 → ✓ (E) 氯乙烯：其中一碳接兩個 H → 相同 → ✗ ⭐ 關鍵：只要有任一端接相同的兩個基，就不存在順反異構物。
Q2
【反應類型判斷】下列反應中，哪一個屬於「取代反應」？ (A) CH₂=CH₂ + Br₂ → CH₂BrCH₂Br (B) CH₄ + Cl₂ → CH₃Cl + HCl（光照） (C) nCH₂=CH₂ → -(CH₂CH₂)ₙ- (D) CH₂=CH₂ + H₂O → CH₃CH₂OH (E) HC≡CH + H₂ → CH₂=CH₂
▼
ANSWER 正確答案：(B) (A) 加成反應：雙鍵打開，Br 加在兩個碳上 ✗ (B) 取代反應：CH₄ 的 H 被 Cl 取代，同時釋出 HCl ✓ (C) 聚合反應 ✗ (D) 加成反應（水合）✗ (E) 加成反應 ✗ ⭐ 取代 = 舊原子「換掉」被「新原子取代」；加成 = 不飽和鍵被打開加入新原子。
Q3
【推測產物】1-丙炔（HC≡C-CH₃）與「過量溴化氫（HBr）」反應，主要產物為何？（提示：依馬可尼可夫法則，每步加成 H 加在氫多的碳上）
▼
ANSWER 步驟①：HC≡C-CH₃ + HBr → CH₂=C(Br)-CH₃ （馬氏：H 加在末端碳，Br 加在中間碳） 步驟②：CH₂=C(Br)-CH₃ + HBr → CH₃-CBr₂-CH₃ （馬氏：H 再加在末端碳） → 主要產物：2,2-二溴丙烷（CH₃CBr₂CH₃） ⭐ 易錯：每步加成都要重新套馬氏法則！不能直接說「Br 都加在同一碳上」。
Q4
【合成路線設計】以乙烷（CH₃CH₃）為起始物，設計最短合成路線得到乙醇（CH₃CH₂OH）。請寫出每步反應及條件。
▼
ANSWER 乙烷不能直接水合，需先脫氫形成乙烯，再水合。 步驟①：CH₃CH₃ → CH₂=CH₂ + H₂ （條件：高溫、觸媒，脫氫反應） 步驟②：CH₂=CH₂ + H₂O → CH₃CH₂OH （條件：H₂SO₄ 催化，加熱，水合加成反應） ⭐ 路線：烷 → 脫氫 → 烯 → 水合 → 醇
Q5
【燃燒分析計算】取一未知有機化合物燃燒，Mg(ClO₄)₂ 增重 1.8 g，NaOH 增重 4.4 g。若分子量為 30，求：(1) 實驗式 (2) 分子式
▼
ANSWER (1) 由 Mg(ClO₄)₂ 增重推算 H 重：1.8 × (2/18) = 0.2 g → H 有 0.2 mol 由 NaOH 增重推算 C 重：4.4 × (12/44) = 1.2 g → C 有 0.1 mol C : H : O = 0.1 : 0.2 : 0 = 1 : 2 : 0 實驗式 = CH₂ (2) 式量 = 14；分子量 30 ÷ 14 ≈ 2 → 分子式 = C₂H₄（乙烯） ⭐ C+H 質量 = 樣品總質量（0.2+1.2=1.4≠?）→ 注意題目數字是否需要查 O 重。
Q6
【芳香烴反應判斷】下列關於苯及其衍生物的敘述，哪些正確？ (A) 苯不能與 KMnO₄ 反應，甲苯可以 (B) 苯與氯在紫外光下進行取代反應得氯苯 (C) 甲苯、乙苯皆可被 KMnO₄ 氧化成苯甲酸 (D) 苯和乙烯均屬不飽和烴，主要進行加成反應 (E) 萘（C₁₀H₈）與蒽（C₁₄H₁₀）互為同分異構物
▼
ANSWER 正確答案：(A)(C) (A) ✓ 苯環不被 KMnO₄ 氧化；甲苯的甲基側鏈→苯甲酸 (B) ✗ 紫外光是「加成」（六氯化苯）；取代需 FeCl₃ 催化！ (C) ✓ 甲苯和乙苯側鏈均被氧化成 -COOH（苯甲酸） (D) ✗ 苯主要進行取代，不是加成 (E) ✗ 萘 C₁₀H₈，蒽 C₁₄H₁₀，分子式不同，非同分異構物
Q7
【末端炔鑑別】甲、乙、丙三種氣體分子式分別為 C₂H₂、C₂H₄、C₂H₆。僅用溴水和銀氨溶液，如何設計鑑別流程？
▼
ANSWER 步驟①：通入溴水 → 乙烯（C₂H₄）和乙炔（C₂H₂）使溴水褪色 → 乙烷（C₂H₆）不反應 → 乙烷鑑別出來 ✓ 步驟②：對剩餘兩種通入銀氨溶液 → 乙炔（末端炔 HC≡CH）產生白色沉澱（乙炔銀 Ag₂C₂） → 乙烯（C₂H₄）不反應 → 兩者鑑別完成 ✓ ⭐ 邏輯：溴水（飽和 vs 不飽和）+ 銀氨液（末端炔 vs 烯）
Q8
【命名練習】寫出以下化合物的 IUPAC 中文名稱： (1) CH₃-CH(CH₃)-CH₂-CH₃ (2) CH₂=C(CH₃)-CH₂-CH₃ (3) HC≡C-CH(CH₃)-CH₃ (4) 苯環上接 -C₂H₅（乙基）
▼
ANSWER (1) 主鏈 4 碳（丁烷），第 2 位有甲基 → 2-甲基丁烷 (2) 主鏈 4 碳（丁烯），含雙鍵在 1 位，2 位有甲基 → 2-甲基-1-丁烯 (3) 主鏈 4 碳（丁炔），含參鍵在 1 位，3 位有甲基 → 3-甲基-1-丁炔 (4) 苯環接乙基 → 以苯為母體 → 乙苯（ethylbenzene）
06

高頻易錯清單

❌
苯使溴水褪色？
苯「不能」直接使溴水褪色！苯的鹵化需要 FeCl₃ 催化才進行取代反應。只有含 C=C 或 C≡C 的烯、炔才能直接使溴水褪色（加成）。
❌
乙炔水合的產物是乙醇？
乙炔（HC≡CH）水合（HgSO₄ 催化）的最終產物是乙醛（CH₃CHO），不是乙醇！中間體烯醇（CH₂=CHOH）不穩定，會重排成乙醛。
❌
聚乙烯（PE）中還有雙鍵？
聚乙烯（PE）中沒有 C=C 雙鍵！聚合後所有雙鍵都被打開，形成 -CH₂-CH₂- 重複單鍵結構，所以 PE 是飽和聚合物。
❌
環烷與烯分子式相同，化性也相同？
環烷（如環己烷 CₙH₂ₙ）與烯同為 CₙH₂ₙ，但化性差異很大！環烷不使溴水或 KMnO₄ 褪色；烯可以。不可只看分子式判斷化性。
❌
苯的碳碳鍵是單雙鍵交替？
苯的碳碳鍵全部等長（1.39 Å），是共振結構，鍵級 1.5，不是真正的單雙鍵交替。苯是平面分子，所有 12 個原子在同一平面，鍵角 120°。
❌
任何炔都能與銀氨液反應？
只有末端炔（R-C≡C-H）才能與銀氨液或氯化亞銅反應！非末端炔（如 2-丁炔 CH₃-C≡C-CH₃）不能。判斷是否有 H 接在參鍵碳上是關鍵。
❌
烷類鹵化的條件是「催化劑」？
烷類鹵化（取代）的條件是光照或高溫，不需要催化劑！這是自由基鏈式反應（光引發），與苯的鹵化（FeCl₃ 催化）完全不同。
❌
甲苯被 KMnO₄ 氧化的產物是甲醛？
甲苯的 KMnO₄ 氧化產物是苯甲酸（C₆H₅COOH），不是甲醛。苯環不被氧化，只有側鏈（-CH₃、-CH₂CH₃ 等）被氧化成 -COOH。
